## 🚀 Overview

**Nexus** is an AI-powered logistics decision-support system designed to predict shipment risks and optimize routing strategies.

By analyzing factors such as weather conditions, traffic congestion, and delays, the system provides actionable insights that improve efficiency and resilience in supply chain operations.

## 🌍 Google Solution Challenge Impact

### Problem

Global supply chains face constant disruptions due to unpredictable factors such as traffic congestion, weather conditions, and operational inefficiencies. These disruptions lead to delays, increased costs, and resource wastage.

### Solution

Nexus provides a decision-support system that predicts shipment risks and suggests optimized routing strategies using AI and simulation.

By analyzing key logistics variables, the system helps users make faster and more informed decisions.

### Impact

* Reduces delays in logistics operations
* Improves efficiency in transportation networks
* Supports smarter infrastructure planning
* Demonstrates how AI can enhance supply chain resilience

* HTML, CSS, JavaScript

* Leaflet.js

---

## 🔄 How It Works

1. User inputs shipment details
2. Backend processes data (AI / simulation)
3. System returns:

   * Risk score
   * Reason
   * Suggestion
4. Frontend visualizes results with charts and maps

## 💻 Setup

### Backend

```bash
cd backend
pip install fastapi uvicorn python-dotenv requests
uvicorn main:app --reload
```

### Frontend

Open `frontend/index.html` in your browser.
